insert(val)[source]¶

Insert data into the BST

Parameters

val – The value to be inserted in the BST

>>> tree.insert(10)
>>> tree.insert(4)
>>> tree.insert(24)
>>> print(tree.root)
10
>>> print(tree.root.left)
4
>>> print(tree.root.right)
24
traverse(order)[source]¶

Traverses the tree in the given order and prints it out

Parameters

order – Specifies the order in which the tree is to be traversed, can be PRE, IN or POST

>>> tree.traverse('PRE')
10 4 24 
>>> tree.traverse('POST')
4 24 10 
>>> tree.traverse('IN')
4 10 24 
>>> tree.insert(64)
>>> tree.traverse('POST') 
4 64 24 10
>>> tree.insert(20)
>>> tree.traverse('PRE') 
10 4 24 20 64

Note that in the first few traversals, I have manually added a trailing whitespace, alternatively doctest flags can also be used as shown

height(root)[source]¶

Gives the height of the tree

Parameters

root – The node from which height is to be found

Returns

The height of the tree from the given node

Return type

int

>>> node = tree.root
>>> print(tree.height(node))
2
>>> print(tree.height(node.left))
0
>>> tree.insert(29)
>>> print(tree.height(node))
3
