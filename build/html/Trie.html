find(root, c)[source]¶

Finds whether the node correctly corresponds to the character

Parameters

  • root – A node of the trie whose character is to be checked

  • c – The character to be checked

Returns

True if they match, else False

Return type

bool

>>> t.insert('banana')
>>> root = t.T
>>> print(t.find(root, 'a'))
False
>>> print(t.find(root, 'b'))
True
>>> print(t.find(root['b'], 'b'))
False
>>> print(t.find(root['b'], 'a'))
True
