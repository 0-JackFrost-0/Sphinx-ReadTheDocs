DSA module¶

DSA.merge(list1, list2)[source]¶

Merges two linked lists together

Parameters

  • list1 – The first list to be merged

  • list2 – The second list to be merged

Returns

Returns the merged Linked list

Return type

SinglyLinkedList

>>> from DSA import merge, SinglyLinkedList
>>> L1 = SinglyLinkedList()
>>> L1.insert(10)
>>> L1.insert(12)
>>> L1.insert(25)
>>> L1.printer()
[10, 12, 25]
>>> L2 = SinglyLinkedList()
>>> L2.insert(67)
>>> L2.insert(100)
>>> L2.insert(26)
>>> L2.printer()
[67, 100, 26]
>>> L3 = merge(L1, L2)
>>> L3.printer()
[10, 12, 25, 67, 100, 26]
